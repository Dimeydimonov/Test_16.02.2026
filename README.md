  Тестовое задание
    
     Описание
PHP модуль с ООП архитектурой для отображения карточек моделей.

    Требования 
PHP - 8.1
PSR-4 автозагрузка классов
Чистая архитектура разделение слоёв MVC

    Структура проекта
 config.php- Конфигурация приложения
 index.php- Точка входа 
 composer.json- Зависимости и автозагрузка
data/
 models.json- Данные моделей

src/
  Config.php- Класс конфигурации
  DataProvider/   - Слой получения данных
  Model/          - Сущности
  Repository/     - Слой работы с данными
  View/           - Слой отображения
  Controller/     - Контроллеры
  templates/      - PHP шаблоны
  public/css/     - Стили
 
    Установка и запуск
1. Клонировать репозиторий:
   (bash
git clone https://github.com/Dimeydimonov/test_16.02.2026.git
cd test_16.02.2026)
2. Установить зависимости:
   (bash
composer install)
3. Запустить встроенный PHP сервер:
   (bash
php -S localhost:8000)
4. Открыть в браузере:
   (http://localhost:8000)

    Архитектура
   DataProvider
   Интерфейс и реализация для загрузки данных из различных источников (JSON, БД).
        Model - Сущность модели с геттерами для всех полей.
        Repository - Преобразует "сырые" данные в объекты Model.
        ViewRenderer - Рендерит PHP шаблоны с переданными данными.
       PageController - Координирует работу всех компонентов и собирает страницу.
